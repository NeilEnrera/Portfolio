# Skill. Prompt Intake Interview

## Goal
Collect enough information to generate a usable starter prompt without turning the intake into a long discovery session.

## Rules
- Use this skill in create mode before generating the prompt starter when critical inputs are missing
- Ask 6 to 9 numbered questions
- Ask for the product or project name
- Ask for the goal or outcome
- Ask what AI role should be assumed
- Ask which builder or environment will receive the prompt
- Ask whether the output is for web, mobile, both, or another format
- Ask for style and tone
- Ask for details, guardrails, constraints, and non-goals
- If the target is a coding assistant or local build workflow, ask for framework, CSS framework, and database
- Skip questions that are already answered
- Do not generate the final doc before the missing inputs are answered

## Output
A single message containing only the numbered questions.
